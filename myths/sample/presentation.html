Hello, World!

This sample demonstrates how this tool can be used to create a slide presentation from a simple YAML file with content formatted using Markdown syntax.

Controls

  • <Space> to go forward
  • p to go back
  • O to open the options menu

Notes can be used to assist you while giving the presentation.

The second slide

This tool is written in Ruby using RDiscount's implementation of Markdown. The slides are generated from a Mustache template.

The next slide will introduce some formatting.

Notes are optional. The next slide has no notes.
